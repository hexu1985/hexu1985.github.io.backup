Arch linux环境配置

Add User

新增用户：


# useradd -m -s /bin/bash hexu
安装sudo，root用户授权：

# pacman -S sudo
修改sudo配置文件 

# vi /etc/sudoers

Install sshd

安装sshd: 


$ sudo pacman -S openssh
启动sshd 

$ sudo systemctl start sshd
开机自启动sshd 

$ sudo systemctl enable sshd

Install net-tools(ifconfig & route)

安装net-tools 


$ sudo pacman -S net-tools bind-tools inetutils
PS: ifconfig,route在net-tools中，nslookup,dig在bind-tools中，ftp,telnet等在inetutils中.

Install bash-completion

安装bash-completion: 


$ sudo pacman -S bash-completion
$ vi ~/.bashrc
增加如下两行 

[[ $PS1 && -f /usr/share/bash-completion/bash_completion ]] && \
    . /usr/share/bash-completion/bash_completion
