Detailed Description

template<class T, typename Compare = std::greater<T>>
class mini_utils::PriorityQueueRef< T, Compare >

维护一个包含数据类型T元素的优先级队列, 优先级队列的keyList由外部维护, 优先级队列只维护keyList的索引的堆, 堆化过程只交换索引, 不交换外部实际元素, 使用类型Compare的比较函数对象

Template Parameters
T元素类型
Compare比较函数对象的类型
Note
当Compare为std::greater(大于)的时候, 数值大的元素优先级高; 当Compare为std::less(小于)的时候, 数值小的元素优先级高; 这点与std::priority_queue正好相反.
Examples
priority_queue_ref/sample_priority_queue_ref1.cpp, and priority_queue_ref/sample_priority_queue_ref2.cpp.
