Detailed Description

template<typename T, typename Compare = std::greater<T>>
class mini_utils::PriorityQueue< T, Compare >

维护一个包含数据类型T元素的优先级队列, 使用类型Compare的比较函数对象

Template Parameters
T元素类型
Compare比较函数对象的类型
Note
当Compare为std::greater(大于)的时候, 数值大的元素优先级高; 当Compare为std::less(小于)的时候, 数值小的元素优先级高; 这点与std::priority_queue正好相反.
Examples
priority_queue/sample_priority_queue1.cpp.
