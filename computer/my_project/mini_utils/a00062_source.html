18 /* class Stopwatch -- simple stopwatch timer for optimization
19 
20  Stopwatch prints out elapsed time information for measuring the run time
21  of long-running tasks. Stopwatch can also return the elapsed time without
22  printing anything.
23 
24  Stopwatch is meant to be used with the RAII idiom. In its
25  simplest use, the block
26 
27  {
28  Stopwatch sw("TheThing()");
29  TheThing();
30  }
31 
32  produces two lines of output like
33 
34  TheThing(): start
35  // anything printed by TheThing()
36  TheThing(): stop 63mS
37 
38  Stopwatch is meant to replace something more complex like
39 
40  {
41  timeval t1, t2;
42  long dt;
43  gettimeofday(&t1, NULL);
44  TheThing();
45  gettimeofday(&t2, NULL);
46  dt = (t2.tv_sec * 1000000 + t2.tv_usec) - (t1.tv_sec * 1000000 + t1.tv_usec);
47  std::cout << "TheThing() took " << dt/1000 << " mS" << std::endl;
48  }
49 
50  What it prints
51  ctor(): "Stopwatch: start"
52  ctor(false): nothing printed
53  show(): "Stopwatch: show at xxxx mS"
54  sets lap time. Get with getLap()
55  stop(): "Stopwatch: stop xxxx mS"
56  sets lap time. Get with getLap()
57  stop() when not running: "Stopwatch: not started"
58  start(): "Stopwatch: start"
59  clears lap time
60  start() when running: "Stopwatch: start xxxx mS"
61  sets lap time. Get with getLap()
62  What it prints when activity name is specified
63  ctor("activity"): "activity: start"
64  show("theEvent"): "activity: theEvent at xxxx mS"
65  stop("theEvent"): "activity: theEvent xxxx mS"
66  sets lap time. Get with getLap()
67  stop("theEvent") not running: "activity: not started"
68  start("theEvent") running: "activity: theEvent xxxx mS"
69  sets lap time. Get with getLap()
70  start("theEvent") not running: "activity: theEvent"
71  clears lap time
72 
73  printing can be suppressed (get time using getLap()
74  ctor("") or ctor(nullptr) prints nothing
75  show("") or show(nullptr) prints nothing, sets lap time
76  start("") or start(nullptr): prints nothing, sets lap time if running
77  stop("") or stop(nullptr): sets lap time. Get with getLap()
