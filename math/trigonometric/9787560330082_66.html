三角函数的连续性: $\cos x$

如果单位圆的点$a$与水平直径的端点, 例如, 与点$A(1,0)$重合, 那么这时就有$\cos a = 1$ 与$\left| \cos x - \cos a \right| = 1 - \cos x < \epsilon$. 如果$\left|x - a \right| < \delta$, 其中只需命$\delta = \arccos (1 - \epsilon) $.

注释:
由于$A(1,0) \Rightarrow a = 2k\pi$,
以及$\left|x - a \right| < \delta \Rightarrow -\delta < x - a < \delta \Rightarrow -\delta < x - 2k\pi < \delta$,
(1)当$-\delta < x - 2k\pi < 0$ (即$2k\pi - \delta < x < 2k\pi$) 时, $\cos x$为增函数, 所以 $\cos (2k\pi - \delta) < \cos x \Rightarrow \cos \delta < \cos x$, 又因为 $\delta = \arccos (1 - \epsilon) \Rightarrow \cos \delta = 1 - \epsilon $, 所以 $1 - \epsilon < \cos x$, 即$1 - \cos x < \epsilon$.
(2)当$0 < x - 2k\pi < \delta$ (即$2k\pi < x < 2k\pi + \delta$) 时, $\cos x$为减函数, 所以 $\cos x > \cos (2k\pi + \delta) \Rightarrow \cos x > \cos \delta$, 又因为 $\delta = \arccos (1 - \epsilon) \Rightarrow \cos \delta = 1 - \epsilon $, 所以 $1 - \epsilon < \cos x$, 即$1 - \cos x < \epsilon$.
